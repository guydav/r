This is documentation for the dataverse replication file for the following article:

Iacus, Stefano M., Gary King, and Giuseppe Porro. "Multivariate Matching Methods That are Monotonic Imbalance Bounding." Journal of the American Statistical Association 106 (2011): 345-361. copy at http://j.mp/lxfJch

Some of the data used in this article is available in the CEM package available at http://gking.harvard.edu/cem and also in CRAN (http://cran.r-project.org/web/packages/cem/).  

The rest is data available at the American Economic Review web site http://www.aeaweb.org/articles.php?doi=10.1257/aer.98.1.311 associated with this article:  Washington, E. L. (2008), “Female Socialization: How Daughters Affect Their Legislator Fathers’ Voting on Woman’s Issues,” American Economic Review, 98 (1), 311–332.
